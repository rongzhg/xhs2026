## 📊 技术栈

- **Framework**: Flask 2.3+
- **数据存储**: JSON文件（可扩展）
- **数据模型**: Python OOP设计
- **爬取库**: xhs（现有项目库）

### 前端
- **框架**: HTML5 + CSS3 + JavaScript
- **UI库**: Bootstrap 5.3
- **图表**: Chart.js 4.4
- **图标**: Bootstrap Icons

### 部署
- **开发**: Python venv
- **生产**: Gunicorn + Nginx
- **容器化**: Docker支持
